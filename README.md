# Harikrishna Sureshkumar Pillai - Professional Portfolio

An editorial-style, multi-section portfolio website built with Next.js and Tailwind CSS, focusing on professional systems thinking and technical correctness.

## Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Typography**: [Inter](https://fonts.google.com/specimen/Inter) (Headings), Serif-based system fonts (Body)
- **Content**: Data-driven JSON architecture
- **Deployment**: Optimized for [Vercel](https://vercel.com/)

## Features
- **Narrative-driven Experience**: Roles framed as short case studies.
- **Conceptual Projects**: Showcasing problem-solving and technical strategy.
- **Editorial Readiness**: Medium-inspired typography and whitespace.
- **Resume Viewer**: Integrated PDF viewer and download functionality.

## Local Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/harikrishna-pillai/harikrishna-portfolio.git
   cd harikrishna-portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to see the result.

## GitHub Deployment

To make this available as a GitHub repository, follow these steps:

1. **Create a new repository** on GitHub named `harikrishna-portfolio`.
2. **Push the local repository**:
   ```bash
   git remote add origin https://github.com/YOUR_GITHUB_USERNAME/harikrishna-portfolio.git
   git branch -M main
   git push -u origin main
   ```

## Deployment

### Vercel (Preferred)
Push your code to a GitHub repository and import it into Vercel. The build settings are automatically detected.

### GitHub Pages
Configure your project for static exports in `next.config.ts` and use GitHub Actions for deployment.
